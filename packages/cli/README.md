# @bit-ocean/config

![npm](https://img.shields.io/npm/v/@bit-ocean/cli?logo=gnubash&label=cli)

> Universal CLI.

This package provides a universal CLI for all our projects.

## Installation

### Global

```bash
pnpm add -g @bit-ocean/cli
bo -v
```

### Project

```bash
pnpm add -D @bit-ocean/cli
npx bo -v
```

## Usage

```bash
bo -help
```
